⚠️ PARTE ADICIONADA PARA A BASE DE DADOS COMPLETA¶

Os dados consideram todas as doses aplicadas. Para descobrir o total de pessoas vacinadas, preciso separar por dose.

In [ ]:
# Vacinados com a primeira dose
primeiraDose = len(vacinacao_DF[vacinacao_DF['vacina_descricao_dose'] == '1 Dose'])

# Vacinados com a dose nica
doseUnica = len(vacinacao_DF[vacinacao_DF['vacina_descricao_dose'] == 'nica'])

# Exibe os valores
print(f'Vacinados com a 1 Dose: {primeiraDose}')
print(f'Vacinados com a Dose nica: {doseUnica}')

Vacinados com a 1 Dose: 2476758
Vacinados com a Dose nica: 395

Considerando que, para receber a segunda dose e as demais doses de reforo, a pessoa deve ter tomado a primeira dose ou a dose nica, podemos considerar que a quantidade indviduos vacinados no DF :

In [ ]:
individuosVacinados = primeiraDose + doseUnica
print(f'Total de indivduos vacinados no DF: {individuosVacinados}')

Total de indivduos vacinados no DF: 2477153

Portanto, 2.477.153 pessoas se vacinaram no DF.
